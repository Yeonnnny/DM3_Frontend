텍스트 관련 스타일 (font)

  • font-family : 글꼴 지정 - 대중글꼴 사용, 웹폰트 사용(구글폰트)
  • font-size : 글꼴 크기 지정
  • font-style : 글꼴 꾸밈(이탤릭체) 지정
  • font-style : 글꼴 꾸밈 지정
  • font-weight : 굵기 (100~900까지 100씩 늘어나도록 : 400 기본), bold, bolder 등
  • font-varient : 영문기준 소문자 크기의 대문자로 표현

텍스트 관련 스타일 (text)

  • color : 글꼴 색상 지정
  • text-decoration : 글꼴 꾸밈(underline, overline, line-through, none)
  • text-transfrom : 영문자일 경우 대소문자 지정
  • text-shadow : 글꼴에 그림자 지정
  • white-space : 공백 처리 방법 지정
  • letter-spacing, word-spacing : 자간 조정
  • text-align : 좌우 정렬방법, left, right, center,...
  • text-justify : 글의 내용을 양 끝으로 어떻게 정렬할 것인가 지정
  • text-indent : 들여쓰기
  • line-height : 줄간 지정
  • text-overflow : overflow 속성과 함께 사용하며, 지정된 영역을 넘치는 콘텐츠 표현을 어떻게 할 것인가 지정
