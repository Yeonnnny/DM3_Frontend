CSS 선택자

요소에 서식이나 배경을 적용하기 위해서는 적용하려는 요소를 정확히 선택할 수 있어야 한다.
선택자는 이러한 요소들을 다영한 방식으로 선택할 수 있도록 해준다.

  • 단순 선택자
    • 전체 선택자 (*) : 전체 요소를 선택할 때 사용 (특잇값 : 0)
    • 태그명 선택자 : 태그이름으로 선택할 때 사용 (특잇값 : 1)
    • 그룹 선택자 : 콤마(,)를 이용해서 동일한 스타일 정의를 가진 모든 HTML 요소를 선택할 때 사용
    • 클래스 선택자 : class라는 속성으로 지정한 값을 가진 모든 요소들을 선택할 때 사용 (클래스는 같은 값을 여러 요소에 사용가능하므로 복수 선택 가능) (특잇값 : 10)
      ex : <p class="one test"> 공백을 기준으로 복수 클래스 선택함. 특이도가 같은 경우는 style에서 아래 쪽에 지정한 것이 우선이됨.
    • 아이디 선택자 : id라는 속성으로 지정한 값울 가진 하나의 요소를 선택할 때 사용 (아이디는 값을 하나의 요소에 사용하므로 단일 요소 선택) (특잇값 : 100)
    • (예) 단순 선택자의 사용 예
  • 가상(Pseudo) 클래스 선택자
    • 참고 : w3schools 가상클래스
    • 가상클래스는 태그나 특정 다른 선택자 뒤에 콜론(:)을 붙여 선택 기능을 추가하는 키워드로 선택한 요소가 특별한 상태여야 만족되는 클래스이다.
    • 주로 a 태그의 상태를 나눠서 가상클래스를 적용했으나, 다른 태그에도 사용할 수 있다.
    • (예) 가상클래스 선택자의 사용 예
      (연습) 테이블에 가상클래스 선택자 사용하기
  • 가상(Pseudo) 요소 선택자
    • 참고 : w3schools 가상요소
    • 가상요소는 태그의 지정된 부분에 스타일을 적용할 때 사용
      가상요소는 콜론 2개(::)를 붙여 요소의 일부분에 스타일을 적용할 수 있다.
      예를 들어 요소의 첫글자, 요소의 첫 줄, 콘텐츠 앞이나 뒤에 또다른 콘텐츠 삽입 등
      (예) 가상요소 선택자의 사용 예
  • 속성 선택자
    • 특정 속성이나 속성값을 가진 HTML 요소를 선택할 때 사용
    • [속성]
      [속성=값]
      [속성~=값]
      [속성|=값]
      [속성^=값]
      [속성$=값]
      [속성*=값]
    • 참고 : w3schools 속성 선택자
    • (예) 속성 선택자의 사용 예
  • 종속 선택자 : 둘 이상의 선택자를 조합해서 사용하는 것
      4가지 주요 조합
    • 공백(후손 선택자) : 하위항목(후손) 선택
    • >(자식 선택자) : 인접한 하위 항목을 선택
    • + (형제 선택자) : 같은 상위 클래스를 가진 연달은 다음 요소
    • ~ (일반 형제 선택자) : 같은 상위 클래스를 가진 요소들
    • (예) 종속 선택자의 사용 예
  • 기타
    • not(선택자) : 괄호 안에 지정한 선택자가 아닌 요소 선택
    • nth-child(수식) : 괄호 안의 수식에 맞는 요소 선택
    • (예) 기타 선택자의 사용 예
